## Tujuan

Repo ini dibangun untuk memudahkan kawan-kawan untuk mencari tutorial pemrograman fungsional dan mempelajarinya secara mendalam.

Serta, demi kebebasan informasi untuk edukasi dan pencerdasaan bersama.

```

```
